BLADE'NHAMMER

Game Info

　

※画面は開発中のものになります

  • ジャンル：アクションゲーム
  • プレイ人数：1人
  • キャッチコピー：わちゃわちゃ振って楽しい
  • ハード：PC
  • コントローラー：Joy-Conまたはマウス
  • チーム：Team Munch-β-

コメント

右ジョイコンを振って攻撃！左ジョイコンで移動や回避！両手にジョイコンを握って、フィジカルな操作でエネミーを倒していくゲーム、それが「Blade’nHammer」！
強力な必殺技やジョイコンの振り方によって異なる攻撃方法を駆使し、登場するさまざまなエネミーを攻略していきます。
「アクションゲーム」ならの疾走感をイメージし、わちゃわちゃしたゲーム体験ができます。
このゲームの特徴として、名前の由来でもあるブレード「剣」とハンマー「槌」の交互に切り替えてエネミーと戦うことができます。それぞれの武器でしか倒せないエネミーや気に入った武器を楽しむことができます。

潮またぎのアルフレッド

Game Info

  • ジャンル：シューティングアクション
  • プレイ人数：1人
  • キャッチコピー：撃ちまくれ逆境を乗り越え海を行け
  • あらすじ
    とある海底で暮らすテッポウエビの「アルフェ」
    最近、人間が捨てたごみで汚れてきていた。いくら片付けてもきりがない。
    仕方がないのでもっと遠くに引っ越すことになった。
    巣穴を出たアルフェは新天地を求め、旅に出ることになったのだが、どうも海の様子がおかしい。
    みんなやけに凶暴だし、天敵のタコやイカが多すぎる。
    無事新しい住処を見つけることができるのだろうか・・・。
  • ハード：PC
  • コントローラー：マウス
  • チーム：GAF-fly

コメント

このゲームは、この主人公のアルフェを操作して無事に目的地まで突破するゲームです。
操作方法は簡単でマウス操作のみ。左クリックで衝撃波を発射してその反動で移動、攻撃ができ、右クリックでその場で一定時間とどまることができます。ぜひ、遊んでみてください！

爆走！！トレイルロード

Game Info

  • ジャンル：3Dタイムアタックゲーム
  • プレイ人数：1人
  • キャッチコピー：危険な坂を誰より早く駆け抜けろ！
  • ハード:PC
  • コントローラー：Joy-Con, PCキーボード
  • チーム：タンメン谷口

コメント

トレイルとは、舗装されていない坂を駆け抜ける危険な行為だ！
このゲームでは、危険な山道や急坂を誰よりも速く駆け下ってもらう！山道にはスピードが落ちてしまう泥や様々な障害物、爽快なジャンプを決められるジャンプ台などいろんなものが転がっているぞ！
こいつらを時にはかわし、時には駆使して最速でトレイルタイムアタックを制せ！！
何度もやるうちに未開の地ならではの抜け道なんかが見つかったりするかも...？
